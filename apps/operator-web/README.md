# Operator Web App

Next.js + Tailwind operator workbench for deterministic operational control.

Pages:
- Overview
- Field Intake
- Boxes / Queues / Shipments
- Grading Workbench
- Analytical Results
- Pricing / Exposure
- Financial Ledger
- Reconciliation
- Settlement list + detail drilldown
- Audit / Evidence Explorer

Runtime:
- defaults to control API at `http://localhost:3001`
- set `DCS_API_BASE_URL` to override
- run `npm run dev --workspace @dcs/operator-web`
